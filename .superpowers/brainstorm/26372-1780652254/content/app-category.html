What kind of Mac app?

Pick the category that fits your idea — or describe something different in the terminal

Productivity / Workflow

Saves time or reduces friction in daily work — launchers, clipboard managers, window managers, focus timers

B

Menu Bar Utility

Lives in the Mac menu bar — system stats, quick actions, status monitors, toggles

Developer Tool

Built for developers — API testers, log viewers, SSH managers, code snippets

Creative / Media
